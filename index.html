Setup Guide - Getting Your GitHub Token

A GitHub Personal Access Token is like a password that lets this app save your projects to your GitHub account. Follow these steps to create one:

1

Go to GitHub Token Settings

Click the button below to open GitHub's token creation page. You'll need to be logged into GitHub.

Open GitHub Token Page
2

Fill in the Token Details

Note: Enter Game Editor (or any name you'll remember)
Expiration: Select No expiration (recommended) or 90 days
3

Select Scopes (Permissions)

Scroll down to "Select scopes" and check ONLY this one:

gist - Create gists

Leave all other checkboxes unchecked for security!

4

Generate Token

Scroll to the bottom and click the green "Generate token" button.

5

Copy Your Token

GitHub will show you your new token (starts with ghp_). Click the copy button next to it.

Important: You won't be able to see this token again! Copy it now.

6

Paste in Game Editor

Come back here and paste your token in the input field, then click "Connect to GitHub".

Frequently Asked Questions

Is my token safe?

Yes! Your token is stored only in your browser's local storage. It's never sent anywhere except directly to GitHub's official API.

What if my token expires?

Your projects are stored on GitHub, not in the token. Just create a new token and all your projects will be there!

What does the "gist" permission do?

It allows the app to create, edit, and delete Gists (code snippets) on your GitHub account. Your projects are stored as Gists.

Can I revoke the token later?

Yes! Go to GitHub Token Settings and click "Delete" next to your token.
